# Система складского учёта АБЗ ВАД

React фронт-приложение для управления складом асфальтобетонного завода ВАД с поддержкой трёх ролей пользователей: Складовщик, Менеджер и Администратор.

## Функциональность

### Для Складовщика (warehouseman):
- **Просмотр товаров** - видит список всех товаров на складе с местоположением и количеством
- **Просмотр заявок** - видит активные заявки по его сегментам
- **Уведомления о низких запасах** - оповещение о товарах с недостаточным количеством

### Для Менеджера (manager):
- **Обзор запасов** - общая статистика по товарам и стоимости
- **Управление заявками** - создание, одобрение и отслеживание заявок
- **Управление местоположениями** - просмотр товаров по зонам склада
- **Отчёты** - создание отчётов о товарах и движениях

### Для Администратора (admin):
- **Управление пользователями** - создание, редактирование, удаление пользователей
- **Логи системы** - просмотр всех действий пользователей в системе
- **Управление площадкой** - настройка параметров склада
- **Настройки системы** - конфигурация безопасности и логирования

## Аутентификация

Приложение включает систему логина и регистрации с тремя ролями. 

### Тестовые учётные данные:
```
Логин: warehouseman1 | Пароль: password | Роль: Складовщик
Логин: manager1       | Пароль: password | Роль: Менеджер
Логин: admin          | Пароль: password | Роль: Администратор
```

## Заметки о реализации

### Mock данные
Все данные в настоящее время хранятся в памяти приложения (`mockService.ts`). Функции сервиса имитируют задержки API для более реалистичного опыта.

### Местное хранилище
Сессия пользователя сохраняется в `localStorage` для сохранения состояния при перезагрузке страницы.

### Маршруты по ролям
Каждая роль имеет доступ к соответствующей панели управления и функциям через компонент `ProtectedRoute`.
